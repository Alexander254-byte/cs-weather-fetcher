# CS Weather Fetcher
Fetches real-time weather using Python + requests and OpenWeatherMap API, now with a Tkinter GUI.

**Run:** `python weather.py`

**Requirements:** 
- Python 3.13
- `pip install requests`
- Free API key from [openweathermap.org](https://openweathermap.org/)
